WEATHERFOR- Real-Time Weather Prediction Web Application

WeatherFor is a cutting-edge web application developed to provide real-time weather predictions for the next hour through advanced data analysis techniques. This project demonstrates my proficiency in Python, Django, data analysis, and UI/UX design. It showcases my ability to create data-driven applications that serve a practical purpose while also employing innovative solutions for improved user interaction. The continuous improvement model reflects my commitment to delivering dynamic and high-impact solutions.

Order tracking app for an art studio - UI/UX design

In this project, I embarked on designing a user-centric Order Tracking App tailored for an art studio, aimed at enhancing customer experience and streamlining operations. Beginning with low-fidelity wireframes and prototypes, I laid the foundation for intuitive user journeys and efficient functionalities. As the project evolved, I transitioned to developing high-fidelity mockups, infusing visual flair while maintaining a focus on usability. Through iterative design processes, I strived to create a seamless interface that empowers users to effortlessly monitor their orders and engage with the art studio's services.
